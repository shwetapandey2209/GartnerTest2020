You need Visual Studio and Git installed
Clone this repository
Open the Solution file and Build the Solution.
Start the Application.
Now,  we can pass below commands to import the files. This app currently accepts two different commands that are intended to import from two different sources, and here are the commands accepted:
The first argument is the type of import, that can be one of the two options: ['capterra', 'softwareAdvice']. The second argument is the location of the file.

softwareAdvice feed-products/softwareAdvice.json

capterra feed-products/capterra.yaml
